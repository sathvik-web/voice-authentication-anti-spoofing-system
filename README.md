# Voice Authentication Anti-Spoofing System

AI-based voice authentication system that verifies speaker identity and detects spoofed or deepfake audio.

## Features
- ECAPA-TDNN speaker verification
- Multi-feature anti-spoof detection
- Audio quality validation (SNR, RMS)
- Spectrogram visualization
- Authentication logging

## Technologies
- Python
- PyTorch
- SpeechBrain
- Librosa
- Torchaudio
